# Python

> ## PyEnv
>
> ### Versiones: Listar, Instalar, Gestionar
>```bash  
>   # Listar
>   pyenv install -l
>```
>
>
> ### Entornos: Listar, Crear, Gestionar
> ```bash
>    # Listar entornos virtuales
>    pyenv virtualenvs
>
>    # Crer entorno virtual 
>    pyenv virtualenv <name>
>
>    # Borrar entorno virtual
>    pyenv uninstall <name>
> ```
